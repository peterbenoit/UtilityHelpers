UtilityHelpers Documentation

The UtilityHelpers class is a collection of utility functions designed to simplify common tasks in JavaScript development. It serves as a versatile toolkit for handling various operations that developers frequently encounter, providing solutions for tasks related to images, arrays, strings, dates, object manipulation, and more. The class reduces the need to repeatedly write boilerplate code by offering ready-made, reusable methods.

For instance, it includes methods to get image details (getImageDetails) and calculate the average color of an image, which can be useful in media-heavy applications. There are also array manipulation methods like shuffleArray, flattenArray, and countUniqueElements, which assist with data transformation and analysis.

String operations like capitalizeWords, pluralize, and escapeHtml help developers format and sanitize text inputs more effectively. Additionally, the class offers solutions for working with dates and times, including randomDateBetweenTwo and timeStampID, which simplify generating timestamps and random dates within ranges.

Another key feature of UtilityHelpers is its support for performance optimization through methods like debounce and throttle, which control the frequency of function execution. Moreover, methods like deepMerge allow for complex object merging, while numberToWords converts numeric values to readable textâ€”handy for applications that display data in a more human-friendly way.

In short, UtilityHelpers provides a centralized and efficient way to manage a wide variety of tasks. It enhances productivity by streamlining repetitive coding challenges, allowing developers to focus on core application logic without reinventing common utility functions.

`capitalizeWords(text)`

Capitalizes the first letter of each word in a string. 

const capitalized = UtilityHelpers.capitalizeWords('hello, world!');

`getImageDetails(imageSrc, calculateAverageColor)`

Fetches basic image details (like width, height, size, and format) from the provided image source URL. Optionally calculates the average color of the image. 

UtilityHelpers.getImageDetails('https://example.com/image.jpg', true)
  .then(details => console.log(details));

`getImageOrientation(image)`

Determines the orientation (landscape, portrait, square) of an image based on its dimensions. 

const orientation = UtilityHelpers.getImageOrientation(image);

`formatBytes(bytes, decimals)`

Formats a size in bytes into a human-readable string. 

const formattedSize = UtilityHelpers.formatBytes(1024);

`getAverageImageColor(imageSrc)`

Calculates the average color of an image from its URL. 

UtilityHelpers.getAverageImageColor('https://example.com/image.jpg')
  .then(color => console.log(color));

`generateUUID()`

Generates a random UUID. 

const uuid = UtilityHelpers.generateUUID();

`shuffleArray(array)`

Randomly shuffles the elements in an array. 

const shuffled = UtilityHelpers.shuffleArray([1, 2, 3, 4]);

`flattenArray(arr)`

Flattens a nested array into a single array. 

const flatArray = UtilityHelpers.flattenArray([1, [2, 3], [4]]);

`sumArray(arr)`

Returns the sum of all values in an array. 

const sum = UtilityHelpers.sumArray([1, 2, 3]);

`deepMerge(target, source)`

Performs a deep merge of two objects.

`pluralize(word, count)`

Attempts to pluralize the given word, accounting for irregular plurals.

`debounce(fn, delay)`

Debounces a function, ensuring it only fires after a specified delay. 

const debouncedFn = UtilityHelpers.debounce(() => console.log('debounced!'), 300);

`textToSpeech(text)`

Converts text to speech if the browser supports the SpeechSynthesis API.
